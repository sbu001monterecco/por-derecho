# Por Derecho — Netlify Static Site Pack

Static bilingual institutional website for **Por Derecho — iniciativa fundacional en formación / foundation initiative in formation**.

## Contents

- `index.html` — Home
- `origen.html` — Origin
- `que-somos.html` — What We Are / What We Are Not
- `programas.html` — Programmes
- `gobernanza.html` — Governance
- `preparacion-institucional.html` — Institutional readiness
- `estatutos.html` — Draft statutes
- `faq.html` — FAQ
- `contacto.html` — Contact via LinkedIn only
- `aviso-legal.html` — Legal notice
- `privacidad.html` — Privacy
- `styles.css` — Site styles
- `main.js` — language toggle only; no analytics, cookies, forms or tracking
- `netlify.toml` — basic Netlify settings
- `404.html` — Not found page

## Important operational constraints

This site is static and intentionally does not include:

- donations or payment links
- document uploads
- case intake
- legal advice form
- whistleblowing channel
- user accounts
- analytics
- tracking cookies

## Deployment to Netlify

1. Log in to Netlify.
2. Choose **Add new site**.
3. Use **Deploy manually** or drag-and-drop the unzipped folder.
4. Netlify will publish the site and provide a temporary Netlify URL.
5. Add a custom domain later if desired.

## Legal note

The text is a working institutional draft only. v0.3 includes a Spanish working draft of the proposed statutes and an English working version for international readability. It should be reviewed by Spanish foundation/notarial/tax counsel before any formal use as a registered foundation site.
